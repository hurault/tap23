OCaml files to run the code generated by the Coq proof (cf directory Coq).

Files :
- explain.ml
  Interface Classif : 
    findAXpCXp.mli generated by Coq 
    + friendly version of findAXp and findCXp
  Module AxpCxp : 
    findAXpCXp.ml generated by Coq 
    + the axioms to be realized in the extracted code: nb_feature T_eq_dec lambda nu T
    + data conversion
- dataset.ml 
    Interface Dataset : all the information needed to run the algorithms for a given dataset
    Module Pokemon, Placement, CarEvaluation, Heart : four implantation for the datasets and models used for experiments (see XGBoost directory)
- runAxpCxp.ml
    File to run findAXp and findCXp for all the instances of a dataset and a model generated using XGBoost (see XGBoost directory)
    Print the results in a file in the "results" directory (the directory should be created before runing the examples)

Usage :
- dune utop
- In utop :
    open Certxp;;
    open Dataset;;
    open RunAXpCXp;;
    module DRun = AllAXpCXp(Heart);;
    DRun.run_all();;
- clean : dune clean
OR
- compiling with ocamlopt : ./do.sh compil
- runing
   ./runAXpCXp placement
   ./runAXpCXp heart
   ./runAXpCXp pokemon
   ./runAXpCXp car
- clean : ./do.sh clean
